# Zorvyn Financial Dashboard

A finance dashboard assignment built in small, production-style feature slices so each milestone can become its own Git commit.

## Current Progress

1. Dashboard foundation and overview
2. Transactions experience
3. Role-based transaction management
4. Insights, persistence, and documentation polish

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

```bash
npm run dev
npm run build
npm run preview
```
